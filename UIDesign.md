# CP 档案站 · UI 设计规范

**版本：** v1.0  
**基于：** 四张页面参考设计图（首页 / 人物简介 / 时间轴 / 大事记）  
**日期：** 2026-05-14

---

## 一、设计风格总述

### 1.1 整体风格定位

| 维度 | 描述 |
|------|------|
| 风格关键词 | 轻量、细腻、少女感、内容导向 |
| 视觉基调 | 浅色系为主，白底微紫调，整体干净通透 |
| 情感气质 | 温柔浪漫，带有轻微的二次元/国风插画质感 |
| 设计语言 | 现代简约 + 柔和圆角 + 插画图片驱动 |

### 1.2 核心设计原则

1. **内容优先**：界面克制，插画封面和人物图是视觉主角，UI 元素不抢戏
2. **轻量层次**：不使用重阴影或粗边框，通过背景色差异和微阴影区分层级
3. **紫色一致性**：主色紫贯穿所有交互态、选中态、强调元素，保持品牌一致
4. **圆润友好**：所有容器、标签、按钮均采用圆角，营造温和氛围

## 二、色彩系统

### 2.1 主色调

```
主色（紫）     #7B5EA7  ← 导航选中、主要按钮、强调文字
主色浅变体     #9B7FCA  ← 悬停态、标签填充
主色极浅背景   #F3EFFA  ← 选中标签底色、浅紫背景块
```

### 2.2 背景色层级

```
页面底色       #F7F6FA  ← 全站背景，极浅紫灰白
内容卡片背景   #FFFFFF  ← 纯白，与底色形成轻微层次
侧栏/区块背景  #F9F8FC  ← 比底色略白，用于侧边栏容器
```

### 2.3 文字色阶

```
主标题         #1A1A2E  ← 深色，接近黑色，略带蓝紫调
正文           #3D3D5C  ← 中深灰紫，主要内容文字
次要文字       #8A8AA0  ← 灰紫，副标题、日期、注释
占位/禁用文字  #C4C4D4  ← 极浅灰紫
```

### 2.4 事件重要性色彩体系

时间轴使用五级重要性颜色标识，贯穿标签徽章与轴线节点：

| 等级 | 标签文字 | 颜色 | 用途说明 |
|------|----------|------|----------|
| 极重要 | 极重要 | `#7B5EA7`（紫） | 推动关系转折的关键事件 |
| 很重要 | 很重要 | `#F5A623`（橙） | 对关系发展有重大影响 |
| 重要 | 重要 | `#5B9BD5`（蓝） | 重要的推进或转折事件 |
| 一般 | 一般 | `#9B9BB0`（灰） | 普通事件或日常互动 |
| 轻微 | 轻微 | `#7DC18A`（绿） | 背景补充或细节事件 |

### 2.5 功能色

```
成功/确认     #7DC18A  ← 绿色
警告          #F5A623  ← 橙色
危险/删除     #E05A5A  ← 红色
链接/跳转     #7B5EA7  ← 同主色
```

### 2.6 边框与分割线

```
卡片边框      无边框（依赖背景色差异）
分割线        #EBEBF0  ← 极浅灰，仅用于需要明确分隔的区域
输入框边框    #D8D8E8  ← 浅灰紫
```

## 三、字体规范

### 3.1 字体家族

| 用途 | 字体 | 备注 |
|------|------|------|
| 中文主字体 | 系统默认无衬线（-apple-system / PingFang SC / Microsoft YaHei） | 优先系统字体保证渲染质量 |
| 英文/数字 | Inter / SF Pro | 数字显示干净精准 |
| 特殊标题（可选） | 自定义字体上传 | 站长可在设置中替换 |

### 3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 超大标题 | 28px | 700 | 1.3 | 首页 Hero 区 Slogan |
| 大标题 | 22px | 700 | 1.4 | CP 名称主标题 |
| 中标题 | 18px | 600 | 1.4 | 章节标题、卡片主标题 |
| 小标题 | 15px | 600 | 1.5 | 事件标题、Tab 文字 |
| 正文 | 14px | 400 | 1.7 | 事件描述、简介文字 |
| 辅助文字 | 13px | 400 | 1.6 | 日期、章节标注、副标签 |
| 极小标注 | 12px | 400 | 1.5 | 提示文字、图例说明 |

### 3.3 特殊文字样式

- **首页 Slogan 强调词**（"心动"）：主色紫 `#7B5EA7`，字重 700，与周围文字形成色彩对比
- **CP 名称副标**（"谢怜 × 花城"）：辅助文字色，字号 13px，位于主标题下方
- **Tab 选中态**：主色紫 + 下划线，字重 600
- **Tab 未选中态**：次要文字色，字重 400

## 四、间距与布局系统

### 4.1 基础间距单位

采用 **4px 基准网格**，所有间距为 4 的倍数：

```
xs:  4px   ← 图标与文字间距、标签内边距垂直向
sm:  8px   ← 行间紧凑间距
md:  12px  ← 标签内边距水平向、列表项间距
lg:  16px  ← 卡片内边距、区块间距
xl:  24px  ← 模块间距
2xl: 32px  ← 大区块间距
3xl: 48px  ← 页面级区块分隔
```

### 4.2 页面整体布局

**首页：**
- 最大宽度：约 960px，水平居中
- 三列等宽卡片网格，间距 16px
- 顶部导航栏固定，背景白色

**CP 详情页：**
- 左侧侧边栏：约 280px（重要事件栏）
- 右侧内容区：剩余宽度（时间轴主区域）
- 两栏间距：24px
- 整体内边距：水平 24px，垂直 20px

### 4.3 内容宽度约束

| 区域 | 宽度 |
|------|------|
| 首页内容区 | max-width: 960px，居中 |
| 详情页总宽 | max-width: 1200px，居中 |
| 侧边栏固定宽 | 280px |
| 搜索框 | 宽度约 480px，居中 |

```
Tab 悬停态：文字变主色紫
```

**首页顶部导航栏：**
```

```

```

圆角：12px

  ┌─────────────────────────┐
  │   封面图（4:3 比例）     │  ← 圆角 12px（仅上方）
```

## 八、动效规范

| 场景 | 动效类型 | 时长 |
|------|----------|------|
| 卡片悬停 | translateY(-4px) + 阴影加深 | 200ms ease |
| Tab 切换 | 下划线滑动 | 200ms ease |
| 标签选中切换 | 背景色渐变 | 150ms ease |
| 展开/折叠事件 | 高度展开 + 透明度渐显 | 250ms ease-out |
| 弹窗/抽屉出现 | 从右侧滑入（侧边面板）或从底部渐显 | 300ms ease-out |
| 页面跳转 | 淡入淡出 | 200ms |
| Toast 提示 | 从顶部滑入，2s 后淡出 | 200ms / 200ms |

## 九、图标规范

- **图标风格**：线性图标（Line Icons），描边 1.5px，圆角端点
- **图标尺寸**：16px（内联）/ 20px（按钮内）/ 24px（独立操作区）
- **常用图标**：← 返回、♡ 收藏/心形、+ 添加、... 更多、∨ 展开、⚙ 设置、☾ 深色模式、📅 日历、⊞ 视图切换
- **心形图标特殊处**：CP 名称旁的心形图标为描边态（未收藏）/ 填充态（已收藏），颜色随状态切换（灰色↔紫色）
