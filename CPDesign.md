## 一、产品概述

### 1.1 产品定位

一款面向 CP 爱好者的关系档案管理工具，支持对每对 CP 进行人物建档、时间轴梳理与大事记记录，兼具检索、个性化定制与内容扩展能力。

### 1.2 目标用户

- 追星/追番/追剧的 CP 粉丝
- 同人/二次创作者需要整理素材
- 个人收藏型用户（私密记录）

### 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| 结构化归档 | 按 CP 维度组织所有内容 |
| 时间线可视化 | 直观呈现关系发展脉络 |
| 高度可定制 | 主题、字体、布局均可自定义 |
| 扩展友好 | 预留插件/模块扩展位 |

## 三、页面详细设计

### 3.1 首页 · CP 分类检索界面

#### 3.1.1 页面布局

```

│  标签筛选:  [全部] [真人] [动漫] [影视] [游戏] … │  ← 分类标签条

│                                                   │
├─────────────────────────────────────────────────┤
│                                                   │
│  ┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐ │  ← CP 卡片网格
│  │ 封面图 │  │ 封面图 │  │ 封面图 │  │   +   │ │

│                                                   │

#### 3.1.2 CP 卡片组件设计

| 元素 | 说明 |
|------|------|
| 封面图 | 用户上传或自定义（支持横版/方版切换） |
| CP 名称 | 大标题，支持 A × B 或 A♡B 等格式 |
| 标签 | 彩色标签徽章（来源分类 + 自定义标签） |
| 状态徽章 | 例：活跃 / 归档 / 完结 |
| 更新时间 | 最近添加事件的时间 |
| 悬停效果 | 卡片上浮 + 显示"快速预览"按钮 |

#### 3.1.3 搜索交互

- 实时搜索（防抖 300ms），搜索范围：CP名称、人物名、标签、事件关键词
- 搜索结果高亮匹配词
- 空结果状态：显示插图 + "暂无相关 CP，[点击创建]"
- 支持高级搜索面板（展开折叠，包含：时间范围、标签多选、状态筛选）

#### 3.1.4 扩展预留

- 标签系统支持用户自定义分类（颜色、名称）
- 卡片视图可切换：网格 / 列表 / 瀑布流
- 支持置顶 / 收藏 / 私密隐藏 CP
- 预留"收藏夹/分组"功能入口

### 3.4 时间轴页（Timeline Tab）

#### 3.4.1 整体交互模式

时间轴采用**垂直滚动轴**布局，支持两种记录模式切换：

```
[简单记录模式] ←→ [详细记录模式]     [+ 添加事件]   [筛选▾] [视图⊞]
```

#### 3.4.2 简单记录模式

```
时间轴竖线
    │
    ◉── 2023-03-15  [初次相遇]                        ← 事件点
    │               一段简短描述（≤100字）
    │               [标签: 相遇] [♡ 3]
    │
    ◉── 2023-05-01  [公开关系]
    │               简短描述
    │               [标签: 里程碑] ★
    │
    ◉── 2023-08-20  [共同出行]
    │               简短描述
```

**事件卡片（简单模式）包含：**
- 日期（精确到日，或仅年月）
- 标题
- 简短描述（纯文本，限制字数）
- 标签徽章
- 重要性标记（★收藏/里程碑）
- 点击展开 → 进入详细记录视图

#### 3.4.3 详细记录模式（事件详情展开）

点击事件后，右侧滑出详情面板（或展开内联）：

```
┌──────────────────────────────────────────┐
│  ← 返回         [编辑✏] [删除🗑] [⋮]   │
│                                           │
│  📅 2023-03-15（周三）                   │
│  ★ 标签: [相遇] [校园]                   │
│                                           │
│  # 初次相遇                              │
│                                           │
│  [富文本正文区域]                        │
│  支持：标题/加粗/斜体/链接/引用/         │
│  图片插入/分割线/列表等格式              │
│                                           │
│  [图片/附件区域]                         │
│  ┌────┐ ┌────┐ ┌────┐ [+添加]           │
│  │图片│ │图片│ │链接│                   │
│  └────┘ └────┘ └────┘                   │
│                                           │
│  来源/出处: [来源输入框]                 │
│  相关人物: [A] [B] [+]                   │
│  关联事件: [选择关联] → 事件连线         │
│  情感强度: ●●●●○                        │
│                                           │
│  [自定义扩展字段区域]                    │
└──────────────────────────────────────────┘
```

#### 3.4.4 时间轴视图选项

| 视图 | 说明 |
|------|------|
| 默认竖轴 | 标准时间线，居中或左对齐 |
| 年份分组 | 按年折叠展示 |
| 双人视角 | A/B 双栏，各自视角事件分列 |
| 日历视图 | 事件映射到日历格上 |
| 密度视图 | 仅显示标题点，快速浏览 |

扩展预留：用户可自定义轴线颜色、节点图标、卡片样式

#### 3.4.5 筛选与搜索

- 按标签筛选
- 按日期范围筛选
- 按重要性筛选（仅看里程碑）
- 关键词搜索事件标题/内容
- 筛选条件可保存为"视图预设"

#### 3.5.1 页面布局

```
┌─────────────────────────────────────────────────┐
│  大事记                  [+ 添加] [筛选] [排序]  │
│                                                   │
│  按年份归档:  [2023 ▾] [2024 ▾] [2025 ▾]        │
│                                                   │
│  ┌────────────────────────────────────────────┐  │
│  │ ★ 2023-03-15   初次相遇                   │  │  ← 里程碑卡片
│  │   这是两人故事的起点...                   │  │
│  │   [相遇] [校园]              [查看详情 →] │  │
│  └────────────────────────────────────────────┘  │
│                                                   │
│  ┌────────────────────────────────────────────┐  │
│  │ 💕 2023-08-01  第一次牵手                 │  │
│  │   ...                                     │  │
│  └────────────────────────────────────────────┘  │
│                                                   │
│  统计面板（可折叠）:                             │
│  总事件数: 42   里程碑: 12   跨越: 2年3个月      │
└─────────────────────────────────────────────────┘
```

#### 3.5.2 里程碑卡片设计

- 图标标记（♡ ★ 🎂 🎉 等，可自定义）
- 日期 + 标题（大号）
- 摘要文本（前2行，点击展开完整内容）
- 标签
- 关联时间轴事件（跳转链接）
- 可设置"周年提醒"（如：每年此日提示）

#### 3.5.3 大事记 vs 时间轴 的关系

- 时间轴：**完整记录流**，所有事件按时序排列
- 大事记：**精华摘取**，从时间轴事件中标记为里程碑，或单独创建
- 两者数据互通，支持一键从时间轴事件"升级为里程碑"

## 四、全局功能设计

### 4.1 自定义样式系统（Theme Engine）

#### 4.1.1 主题配置面板入口

全局设置（右上角⚙）→ 外观定制

#### 4.1.2 可定制项清单

**颜色系统：**

| 配置项 | 说明 |
|--------|------|
| 主色调 | 影响按钮、链接、强调色 |
| 背景色 | 页面背景（纯色/渐变/图片） |
| 卡片背景 | 内容卡片底色 |
| 文字色 | 主文字/次文字 |
| 轴线颜色 | 时间轴线与节点颜色 |
| 标签配色方案 | 每个标签类型独立配色 |

**字体系统：**

| 配置项 | 选项 |
|--------|------|
| 标题字体 | 预设字体库 + 自定义字体文件上传 |
| 正文字体 | 同上 |
| 字体大小基准 | 滑块调节（影响全局相对大小） |
| 行高 | 调节阅读舒适度 |
| 字体粗细 | 标题/正文独立设置 |

**布局系统：**

| 配置项 | 选项 |
|--------|------|
| 内容宽度 | 窄/标准/宽/全屏 |
| 卡片圆角 | 滑块 0px ~ 24px |
| 卡片阴影 | 无/轻/中/重 |
| 间距密度 | 紧凑/标准/宽松 |
| 动画速度 | 无动画/快/中/慢 |

#### 4.1.3 主题预设与管理

- 内置主题：清新白、深夜黑、樱花粉、墨绿雅、复古纸
- 用户可保存自定义主题（命名 + 截图预览）
- 主题支持导入/导出（JSON格式）
- **CP 级主题**：每对 CP 可设置独立主题，进入详情页自动切换

#### 4.1.4 扩展预留

- CSS 变量覆盖（高级用户可直接输入自定义 CSS）
- 主题商店（预留接入外部主题包入口）
- 节日/特殊日期临时主题（如周年纪念日自动切换）

### 4.3 添加/编辑事件 · 基础交互流程

```
点击「+ 添加事件」
        │
        ▼
  ┌─────────────────┐
  │  选择记录类型   │
  │  ○ 简单记录     │  → 弹出轻量输入框（日期+标题+摘要+标签）
  │  ○ 详细记录     │  → 打开完整编辑器页面/抽屉
  └─────────────────┘
        │

        │
  [预览] → [发布/保存]
        │

**编辑器功能：**
- 富文本（支持 Markdown 语法快捷键）
- 图片上传（拖放 + 粘贴）
- 日期输入（支持模糊日期：仅年/仅年月）
- 标签多选 + 快速创建新标签
- 草稿自动保存（本地暂存）
- 版本历史（预留）

## 五、交互规范

### 5.1 导航逻辑

```
首页
  └─→ CP 详情页（点击卡片）
          ├─→ 人物简介 Tab
          ├─→ 时间轴 Tab
          │       └─→ 事件详情（侧滑/展开）
          │               └─→ 编辑事件
          ├─→ 大事记 Tab
          └─→ 自定义 Tab
```

- 所有页面支持面包屑导航
- 浏览器前进/后退正确响应
- 详情页内 Tab 切换不刷新页面（SPA 路由）

### 5.2 状态反馈规范

| 操作 | 反馈方式 |
|------|----------|
| 保存成功 | Toast 提示（右上角，2秒消失） |
| 删除操作 | 二次确认弹窗 + 撤销窗口（5秒） |
| 加载中 | 骨架屏（非 Loading 转圈） |
| 表单错误 | 行内红色提示文字 |
| 空状态 | 插图 + 引导文字 + 行动按钮 |

### 5.3 响应式断点

| 断点 | 布局策略 |
|------|----------|
| ≥1280px（桌面） | 多列网格，侧边面板 |
| 768px~1279px（平板） | 双列网格，Tab 收缩 |
| <768px（手机） | 单列，底部导航，抽屉交互 |

## 七、设计系统基础规范

### 7.1 间距系统

采用 4px 基准网格：4 / 8 / 12 / 16 / 24 / 32 / 48 / 64px

### 7.2 颜色语义

| Token | 用途 |
|-------|------|
| `color-primary` | 主要交互色（按钮、链接） |
| `color-accent` | 强调色（里程碑、重要标记） |
| `color-surface` | 卡片/面板背景 |

## 十一、两大模块联动说明

事件记录操作与账号权限系统在以下三个维度深度联动：

**内容可见性**：事件编辑器的「可见范围」字段直接对应权限系统的可见性级别（公开/成员可见/指定成员/私密）。

**编辑归属**：事件保存时记录「作者」字段，权限系统据此判断编辑/删除操作是否属于「自己的事件」。

**审核流程**：贡献者的提交内容可配置为需经审核才发布，编辑器保存后进入「待审核」状态，由编辑者/管理员处理。

数据结构设计统一，实现时可共用一套权限中间件，避免重复校验逻辑。
