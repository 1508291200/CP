# CP Timeline 启动指南

## 项目概述
CP Timeline 是一个基于 React + Vite + TailwindCSS 的单页应用，用于记录 CP 时间线和相关资料，包含分类搜索界面、CP 详情页（人物简介/时间轴/大事记）等功能模块。

## cp-timeline - CP Timeline

### 快速启动

```bash
cd cp-timeline
npm run dev
```

**启动后访问**：http://localhost:5173

```yaml
subProjectPath: cp-timeline
command: npm run dev
cwd: cp-timeline
port: 5173
previewUrl: http://localhost:5173
description: CP时间线记录网页应用，React + Vite + TailwindCSS + Dexie 技术栈
```
