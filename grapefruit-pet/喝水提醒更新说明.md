# 喝水提醒功能更新说明

## 已解决的问题

### 1. 窗口裁切问题 ✅
**问题描述**：原本的喝水提醒气泡框使用`position: absolute`定位在角色上方，由于窗口很小（100x120px）且透明无边框，导致提醒框被窗口边界裁切，用户无法看到完整内容。

**解决方案**：
- 移除了窗口内的气泡提醒框（WaterReminder组件）
- 改用Electron的系统原生通知（Notification API）
- 系统通知不受窗口限制，会显示在操作系统的通知区域
- 点击通知可直接标记"已喝水"并重置计时

### 2. 倒计时显示优化 ✅
**新增功能**：
- 在角色下方添加了醒目的倒计时指示器
- 显示剩余时间（例如：`45m`、`30s`）
- 时间到达时显示水滴图标 `💧`
- 使用渐变蓝色背景，带有脉动动画效果
- 倒计时自动每秒更新

**样式特点**：
- 渐变背景：`#2196f3` → `#1976d2`
- 圆角设计，带阴影
- 脉动动画，引起注意但不刺眼
- 不干扰拖拽操作（`pointer-events: none`）

### 3. 自定义时间设置 ✅
**新增功能**：
- 在右键菜单的"喝水提醒"子菜单中添加"自定义时间"选项
- 提供预设快捷选项：15分钟、45分钟、180分钟
- 使用系统对话框，用户体验一致

**菜单结构**：
```
💧 喝水提醒
├── ✓ 已喝水（重置计时）
├── ⚙️ 设置提醒间隔
│   ├── 30 分钟
│   ├── 60 分钟
│   ├── 90 分钟
│   ├── 120 分钟
│   └── ⌨️ 自定义时间...
│       ├── 15分钟
│       ├── 45分钟
│       └── 180分钟
└── 🔕 关闭提醒
```

## 技术实现

### 系统通知
```typescript
// 主进程（main.ts）
ipcMain.handle('notification:show', (_, options) => {
  const notification = new Notification({
    title: options.title,
    body: options.body,
    silent: false,
  });
  
  // 点击通知时标记已喝水
  notification.on('click', () => {
    mainWindow?.webContents.send('water:drink');
  });
  
  notification.show();
});
```

### 倒计时显示
```typescript
// App.tsx - 计算剩余时间
const updateRemaining = () => {
  const remaining = intervalMs - timeSinceLastDrink;
  
  if (remaining <= 0) {
    setTimeRemaining('💧');
    return;
  }
  
  const minutes = Math.floor(remaining / 60000);
  const seconds = Math.floor((remaining % 60000) / 1000);
  
  if (minutes > 0) {
    setTimeRemaining(`${minutes}m`);
  } else {
    setTimeRemaining(`${seconds}s`);
  }
};
```

## 使用指南

### 查看倒计时
- 倒计时自动显示在角色下方
- 大于1分钟时显示分钟数（例如：`45m`）
- 小于1分钟时显示秒数（例如：`30s`）
- 时间到达时显示 `💧` 提醒喝水

### 接收提醒通知
- 到达设定时间后，系统会弹出通知
- 通知标题：`💧 喝水提醒`
- 通知内容：`该喝水啦~ 保持活力！`
- 点击通知可直接标记"已喝水"

### 设置提醒时间
1. 右键点击西柚角色
2. 选择"💧 喝水提醒" → "⚙️ 设置提醒间隔"
3. 选择预设时间或"自定义时间"
4. 在对话框中选择所需时间

### 手动标记已喝水
- 右键菜单 → "💧 喝水提醒" → "✓ 已喝水（重置计时）"
- 或点击系统通知

### 关闭提醒
- 右键菜单 → "💧 喝水提醒" → "🔕 关闭提醒"
- 再次点击可重新开启

## 注意事项

1. **系统通知权限**：首次运行时，操作系统可能会询问是否允许应用发送通知，请点击"允许"
2. **工作时间限制**：提醒仅在工作时间内生效（默认：09:00-18:00）
3. **持久化存储**：提醒设置会自动保存，重启应用后仍然生效
4. **倒计时精度**：倒计时每秒更新一次，确保实时显示
