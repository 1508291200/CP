- 无SVG文件

## 💡 技巧提示

### 调整尺寸

编辑 `electron/main.ts`:
```typescript
const WINDOW_CONFIG = {
  width: 100,   // 改这里
  height: 120,  // 改这里
};
```

### 修改颜色

编辑 `src/components/Character.css`:
```css
.grapefruit-body {
  background: linear-gradient(
    135deg,
    #FFB6C1 0%,   /* 改这里 */
    #FF8FA0 50%,
    #FF7A8A 100%
  );
}
```

### 添加新状态

1. 在 CSS 中定义动画
2. 在菜单中添加选项
3. 在 store 中处理状态
