# 🚀 快速启动指南

## 项目已创建！

位置：`f:/YouAI/grapefruit-pet`

## ⚠️ 重要提示

由于当前环境未安装 Node.js，需要先安装以下软件：

### 1. 安装 Node.js

下载地址：https://nodejs.org/

推荐版本：LTS（长期支持版本，当前为 18.x 或 20.x）

### 2. 验证安装

安装完成后，在命令行运行：

```bash
node --version
npm --version
```

应该看到版本号输出。

## 📦 安装项目依赖

```bash
cd f:\YouAI\grapefruit-pet
npm install
```

这会安装所有必要的依赖包（可能需要 5-10 分钟）

## 🎮 启动开发模式

```bash
npm run electron:dev
```

启动后会看到一个**小巧的西柚**出现在屏幕右下角！

## 🎨 项目特点

### 超简洁UI设计

✅ **窗口尺寸**：100x120px（非常小巧）
✅ **透明背景**：与桌面完美融合
✅ **纯CSS绘制**：无需任何图片资源
✅ **流畅动画**：呼吸、眨眼、弹跳等

- 🖱️ 拖拽移动

### 窗口特性

- 始终置顶

## 🎯 使用技巧

1. **移动位置**：直接拖拽西柚到任意位置
2. **更换心情**：右键选择状态
3. **快速退出**：右键 → 退出

## 🐛 常见问题

### Q: 看不到西柚？

A: 检查是否被其他窗口遮挡，可以右键桌面空白处刷新

### Q: 无法拖拽？

A: 确保点击的是西柚本体，不是透明区域

### Q: 如何关闭？

A: 右键西柚 → 退出
