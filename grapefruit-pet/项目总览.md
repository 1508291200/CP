## 🎯 设计理念

### 核心原则：**极简主义**

- ⭐ **超小窗口**：100x120px，不干扰工作
- ⭐ **纯CSS实现**：无图片资源，性能优秀
- ⭐ **透明融合**：与桌面完美结合
- ⭐ **流畅动画**：自然呼吸，生动可爱

## 🎨 视觉设计

### 西柚水果造型（纯CSS）

```css
外观：
├─ 圆形主体（70px）
│  ├─ 渐变粉红色
│  ├─ 果肉纹理（4个半透明圆）
│  ├─ 中心白点
│  └─ 阴影效果
│
├─ 表情系统
│  ├─ 眼睛（2个黑点，会眨眼）
│  └─ 嘴巴（弧形，状态可变）
│
└─ 顶部绿叶（薄荷绿）
```

### 配色方案

| 元素 | 颜色 | 说明 |

| 绿叶 | #A8D8B9 → #8BC9A3 | 薄荷绿渐变 |

## 🎭 动画状态

### 1. 待机状态（Idle）
```
动画：呼吸效果
周期：3秒
效果：scale(1) ↔ scale(1.05)
```

### 2. 开心状态（Happy）
```
动画：上下弹跳
周期：0.6秒
效果：translateY(0) ↔ translateY(-10px)
嘴巴：更大的弧度
```

### 3. 工作状态（Work）
```
动画：左右摇摆
周期：2秒
效果：rotate(-2deg) ↔ rotate(2deg)
```

### 4. 休息状态（Sleep）
```
动画：慢呼吸
周期：4秒
眼睛：闭眼（扁平）
嘴巴：缩小
```

## 🎨 设计哲学

### 少即是多

> "设计不是添加，而是减去不必要的部分"

- 🎯 只保留核心功能
- 🎨 极简视觉设计
- 💨 轻量级实现
- 🚀 快速响应

### 不干扰原则

> "陪伴，但不打扰"

- 📐 超小窗口（100x120px）
- 👻 透明融合
- 🤫 静默运行
- ⚡ 低资源占用
