# 喝水提醒功能完善说明

## 新增功能

### 1. 重复提醒功能 ✅
**功能描述**：当喝水时间到达后，如果用户未标记"已喝水"，系统会每隔2分钟重复弹出提醒通知，直到用户喝水为止。

**实现逻辑**：
```typescript
// 超时后，每2分钟重复提醒一次
const timeSinceLastNotification = now - lastNotificationTime;
const repeatIntervalMs = 2 * 60 * 1000; // 2分钟

if (lastNotificationTime === 0 || timeSinceLastNotification >= repeatIntervalMs) {
  // 显示系统通知
  window.electronAPI.notification.show({
    title: '💧 喝水提醒',
    body: '该喝水啦~ 保持活力！',
  });
  lastNotificationTime = now; // 更新最后通知时间
}
```

**特点**：
- 第一次提醒：时间到达时立即弹出
- 重复提醒：之后每2分钟弹出一次
- 停止条件：用户点击"已喝水"或点击系统通知
- 智能重置：在工作时间外或未超时时自动停止重复提醒

### 2. 自定义时间输入 ✅
**功能描述**：用户可以自己输入任意提醒间隔时间（1-600分钟），不再局限于预设选项。

**使用方式**：
1. 右键点击西柚角色
2. 选择"💧 喝水提醒" → "⚙️ 设置提醒间隔" → "⌨️ 自定义时间..."
3. 在弹出的窗口中输入分钟数（1-600）
4. 点击"确定"保存

**界面特点**：
- 精美的渐变蓝色背景
- 清晰的输入框，支持数字验证
- 自动选中输入内容，方便快速修改
- 支持键盘操作：
  - `Enter` 键确认
  - `Esc` 键取消
- 实时错误提示（输入范围外时）

**输入限制**：
- 最小值：1分钟
- 最大值：600分钟（10小时）
- 只能输入整数

## 使用场景

### 场景1：工作专注时
设置60分钟提醒，工作时不被频繁打扰，时间到后每2分钟温柔提醒一次。

### 场景2：运动恢复期
设置15-30分钟短间隔，及时补充水分，确保不错过。

### 场景3：长时间会议
设置90-120分钟，会议间隙及时补水。

### 场景4：个性化需求
使用自定义时间，输入任意分钟数，完全符合个人习惯。

## 技术实现细节

### 重复提醒逻辑
```typescript
let lastNotificationTime = 0; // 使用闭包保存状态

const checkWaterReminder = () => {
  if (isWorkTime && timeSinceLastDrink >= intervalMs) {
    const timeSinceLastNotification = now - lastNotificationTime;
    const repeatIntervalMs = 2 * 60 * 1000;

    if (lastNotificationTime === 0 || timeSinceLastNotification >= repeatIntervalMs) {
      showNotification();
      lastNotificationTime = now;
    }
  } else {
    lastNotificationTime = 0; // 重置
  }
};
```

## 注意事项

1. **重复提醒间隔固定为2分钟**：这是平衡提醒频率和用户体验的最佳值，不建议修改
2. **检查间隔为10秒**：系统每10秒检查一次，确保不错过提醒时间
3. **工作时间限制**：非工作时间不会弹出提醒，也不会重复提醒
4. **自动停止**：只要标记"已喝水"，所有重复提醒立即停止
5. **输入范围**：1-600分钟的限制是合理的健康范围
