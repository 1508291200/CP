# 灵感记录框尺寸优化 v2

## 🎯 本次更新

### 1. 大幅增加窗口尺寸 ✅

**之前（第一次优化）**：
- 宽度：700px
- 内容区高度：150px (最大400px)

**现在（第二次优化）**：
- 宽度：**1400px** (增加100%，2倍宽)
- 内容区高度：**300px** (增加100%，2倍高)
- 最大高度：**600px** (增加50%)
- 整体最小高度：**600px**

### 2. 优化字体大小 ✅

为了适配更大的窗口，适当缩小字体保持视觉平衡：

```
标题：18px (保持)
标签文字：12px (从13px缩小)
输入框文字：13px (从15px缩小)
按钮文字：13px (从14px缩小)
```

### 3. 修复默认打开问题 ✅

**问题**：
- 应用启动时自动弹出灵感输入框
- 影响用户体验

**解决方案**：
- 修改persist配置，只持久化灵感数据
- UI状态（isQuickInputVisible、isLibraryVisible）不再持久化
- 确保每次启动都是关闭状态

## 🔧 技术细节

### 修改的文件

#### 1. `src/components/QuickInput.css`
```css
/* 窗口尺寸 */
max-width: 700px → 1400px  /* 2倍宽 */
min-height: 新增 600px      /* 确保足够高 */

/* 内容区域 */
min-height: 150px → 300px  /* 2倍高 */
max-height: 400px → 600px  /* 增加50% */
line-height: 1.6 → 1.8     /* 增加行高 */

/* 字体大小 */
.quick-input-title: 16px → 18px
.input-label: 13px → 12px
input/textarea: 15px → 13px
button: 14px → 13px

/* 间距调整 */
gap: 12px → 16px           /* 增加间距 */
```

#### 2. `src/store/inspirationStore.ts`
```typescript
// 添加 partialize 配置
{
  name: 'inspiration-storage',
  partialize: (state) => ({
    inspirations: state.inspirations,
    // 不持久化UI状态
  }),
}
```

**说明**：
- `partialize` 用于选择性持久化
- 只保存 `inspirations` 数据
- UI状态不再保存，确保每次启动都是关闭状态
