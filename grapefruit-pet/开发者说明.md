## 📦 需要安装的依赖

由于系统未安装 Node.js，需要先：

1. **安装 Node.js**
   - 下载：https://nodejs.org/
   - 选择 LTS 版本（推荐 18.x 或 20.x）

2. **安装项目依赖**
   ```bash
   cd f:\YouAI\grapefruit-pet
   npm install
   ```

   这会安装以下包：
   - electron (约50MB)
   - react + react-dom
   - zustand
   - vite + 插件
   - typescript
   - 其他开发依赖

   总下载大小：约 **200-300MB**
   安装时间：约 **5-10分钟**（取决于网速）

## 🚀 启动项目

### 开发模式

```bash
npm run electron:dev
```

这会：
1. 启动 Vite 开发服务器（端口5173）
2. 启动 Electron 窗口
3. 支持热更新（修改代码自动刷新）

### 首次启动可能遇到的问题

#### 问题1：npm 不是内部命令
**原因**：Node.js 未安装或未添加到环境变量

**解决**：
1. 安装 Node.js
2. 重启命令行
3. 运行 `node --version` 验证

#### 问题2：安装依赖很慢
**原因**：npm 默认源在国外

**解决**：使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

#### 问题3：Electron 下载失败
**原因**：网络问题

**解决**：设置 Electron 镜像
```bash
npm config set electron_mirror https://npmmirror.com/mirrors/electron/
npm install
```

## 🎨 修改角色外观

### 修改颜色

编辑 `src/components/Character.css`：

```css
/* 西柚主体颜色 */
.grapefruit-body {
  background: linear-gradient(
    135deg,
    #FFB6C1 0%,    /* 浅粉 - 可修改 */
    #FF8FA0 50%,   /* 中粉 - 可修改 */
    #FF7A8A 100%   /* 深粉 - 可修改 */
  );
}

/* 绿叶颜色 */
.leaf {
  background: linear-gradient(
    135deg,
    #A8D8B9 0%,    /* 薄荷绿 - 可修改 */
    #8BC9A3 100%   /* 深绿 - 可修改 */
  );
}
```

### 修改尺寸

编辑 `electron/main.ts`：

```typescript
const WINDOW_CONFIG = {
  width: 100,   // 窗口宽度 - 可修改
  height: 120,  // 窗口高度 - 可修改
  minWidth: 80,
  minHeight: 100,
};
```

编辑 `src/components/Character.css`：

```css
.grapefruit-body {
  width: 70px;   /* 西柚大小 - 可修改 */
  height: 70px;
  /* ... */
}
```

### 添加新状态

1. 在 `src/components/ContextMenu.tsx` 添加菜单项
2. 在 `src/components/Character.css` 添加动画

```css
/* 新状态示例 */
.character-excited .grapefruit-body {
  animation: shake 0.5s ease infinite;
}

@keyframes shake {
  0%, 100% { transform: translateX(0); }
  25% { transform: translateX(-5px); }
  75% { transform: translateX(5px); }
}
```

## 🎯 性能优化建议

### 当前已优化

✅ 纯CSS动画（GPU加速）
✅ 无图片资源
✅ 最小化DOM节点
✅ 节流处理拖拽事件

### 如果要进一步优化

1. **减少动画**
```css
/* 关闭呼吸动画 */
.grapefruit-body {
  animation: none;
}
```

2. **降低动画帧率**
```css
/* 从 60fps 降低到 30fps */
.character {
  animation-timing-function: steps(2);
}
```

3. **减少阴影效果**
```css
.grapefruit-body {
  box-shadow: none; /* 移除阴影 */
}
```

## 🎨 设计原则提醒

在添加新功能时，请遵循：

1. **保持小巧**
   - 窗口不超过 150x180px
   - 新UI元素尽量小巧

2. **简洁优先**
   - 减少不必要的装饰
   - 纯色 > 渐变
   - 扁平 > 立体

3. **性能至上**
   - CSS > JavaScript
   - 静态 > 动态
   - 简单 > 复杂

4. **不干扰原则**
   - 透明融合
   - 静默运行
   - 快速响应
