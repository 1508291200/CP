# 🎮 西柚桌宠互动指南（更新版）

## 🍊 你的西柚已经启动了！

现在屏幕右下角应该有一个可爱的粉色西柚在轻轻呼吸~

---

## 🖱️ 基础互动

### 1. 拖拽移动 ✅

**操作**：
- 鼠标**左键按住**西柚
- 拖动到任意位置
- 松开鼠标

**效果**：
- 西柚会变半透明（70%透明度）
- 可以移动到屏幕任意位置
- 不会拖出屏幕边界（自动限制）

**试试看**：
```
把西柚拖到屏幕左上角
或者中间
或者你喜欢的任何位置！
```

---

### 2. 双击打开菜单 ✅

**操作**：
- 鼠标**双击**西柚

**会弹出菜单**：
```
┌─────────────┐
│ 🍊 西柚桌宠  │
├─────────────┤
│ 切换状态     │
│  😊 开心     │
│  💼 工作     │
│  😴 休息     │
│  🌸 待机     │
├─────────────┤
│ 🚪 退出      │
└─────────────┘
```

**试试看**：
```
双击西柚 → 点击"开心"
西柚会开始上下弹跳！
```

---

### 3. 切换心情状态

双击菜单中有4种状态可选：

#### 😊 开心状态
- **动画**：上下弹跳
- **嘴巴**：变大，更开心的弧度
- **适合**：完成任务时、心情好时

#### 💼 工作状态
- **动画**：左右轻轻摇摆
- **适合**：专注工作时

#### 😴 休息状态
- **动画**：慢慢呼吸
- **眼睛**：闭上（变成一条线）
- **嘴巴**：缩小
- **适合**：休息时间、午休时

#### 🌸 待机状态
- **动画**：正常呼吸
- **眼睛**：会眨眼（4秒1次）
- **适合**：日常陪伴

## 🎨 观察西柚的细节

### 自动动画

**西柚会自己做这些事**：

1. **呼吸动画**
   - 轻微放大缩小
   - 3秒一个循环
   - 非常自然

2. **眨眼动画**
   - 每4秒眨一次眼
   - 眼睛会变成一条线
   - 然后恢复

3. **果肉纹理**
   - 4个半透明圆圈
   - 模拟真实果肉
   - 中心有白点

4. **顶部绿叶**
   - 薄荷绿渐变
   - 旋转15度角
   - 增添可爱感

## 🎯 互动小技巧

### 1. 找个好位置

**推荐位置**：
- ✅ 屏幕右下角（默认）- 不挡东西
- ✅ 任务栏旁边 - 容易看到
- ✅ 显示器边缘 - 若隐若现
- ❌ 屏幕中央 - 会挡住工作区

### 2. 根据工作状态切换

```
开始工作 → 双击 → 切换到"💼 工作"状态
完成任务 → 双击 → 切换到"😊 开心"状态
休息时间 → 双击 → 切换到"😴 休息"状态
```

### 3. 观察动画

西柚的动画都很细腻：
- 呼吸是缓慢的
- 弹跳是轻快的
- 摇摆是温柔的
- 眨眼是自然的
