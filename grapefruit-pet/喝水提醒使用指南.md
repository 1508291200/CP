# 💧 喝水提醒功能使用指南

## ✅ 功能已添加！

喝水提醒功能已经集成到西柚桌宠中。

---

## 🎯 功能说明

### 自动提醒
- **默认间隔**：60分钟
- **工作时间**：9:00 - 18:00
- **提醒方式**：西柚会出现蓝色气泡提示

### 提醒样式
西柚上方会弹出可爱的提醒气泡：
```
     💧
  该喝水啦~ 💧
  
 [已喝水 ✓] [10分钟后提醒]
```

---

## 🎮 如何使用

### 方式1：点击提醒气泡
当提醒弹出时：
- **点击"已喝水 ✓"** → 重置计时器
- **点击"10分钟后提醒"** → 推迟10分钟

### 方式2：通过菜单
点击西柚右下角的 ⋮ 按钮：

```
💧 喝水提醒 ▶
  ├ ✓ 已喝水（重置计时）
  ├ ⚙️ 设置提醒间隔 ▶
  │  ├ 30 分钟
  │  ├ 60 分钟
  │  ├ 90 分钟
  │  └ 120 分钟
  └ 🔕 关闭提醒
```

---

## 📐 工作时间设置

当前默认工作时间：**9:00 - 18:00**

只在这个时间段内会提醒，避免夜间打扰。

---

## 🎨 状态变化

### 口渴状态
提醒时，西柚会：
- 轻微摇晃
- 嘴巴变成蓝色
- 显示蓝色提醒气泡

### 喝水后
点击"已喝水"后，西柚会：
- 变成开心状态（弹跳）
- 2秒后恢复待机状态
- 计时器重置

---

## ⚙️ 自定义设置

### 修改提醒间隔
菜单 → 喝水提醒 → 设置提醒间隔 → 选择时长

可选：30/60/90/120 分钟

### 临时关闭
菜单 → 喝水提醒 → 🔕 关闭提醒

再次点击可重新开启

---

## 💾 数据持久化

所有设置会自动保存：
- 提醒间隔
- 开启/关闭状态
- 上次喝水时间

重启应用后继续生效！

---

## 🧪 快速测试

想立即看到提醒？

1. **打开开发者工具**（F12）
2. **在控制台输入**：
   ```javascript
   // 设置上次喝水时间为2小时前
   localStorage.setItem('water-reminder-storage', JSON.stringify({
     state: {
       enabled: true,
       interval: 60,
       lastDrinkTime: Date.now() - 2 * 60 * 60 * 1000,
       workHours: { start: '09:00', end: '18:00' }
     }
   }))
   ```
3. **刷新页面**（Ctrl + R）
4. **提醒会立即弹出！**

## 💡 小技巧

### 1. 定时间隔建议
- **办公场景**：60分钟
- **家庭场景**：90分钟
- **健身期间**：30分钟

### 2. 推迟功能
如果正在忙，点击"10分钟后提醒"，不会错过喝水！

### 3. 手动标记
即使提醒还没到，也可以通过菜单手动标记"已喝水"
