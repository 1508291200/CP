# 🔍 右键菜单调试指南

## 问题：右键点击显示系统菜单而不是自定义菜单

### 已修复内容

✅ 移除了 `-webkit-app-region: drag`（会阻止自定义右键菜单）
✅ 使用纯 JavaScript 实现拖拽
✅ 添加了事件阻止传播
✅ 添加了调试日志

---

## 🔄 让修复生效

### 方法一：热重载（推荐）

1. **在 Electron 窗口中按 `Ctrl + R`** 刷新页面

### 方法二：完全重启

1. 关闭 Electron 窗口
2. 在命令行窗口按 `Ctrl + C` 停止进程
3. 重新运行：`npm run electron:dev`

---

## 🧪 测试步骤

### 1. 打开开发者工具

在 Electron 窗口中按 `F12` 或 `Ctrl + Shift + I`

### 2. 右键点击西柚

在控制台(Console)中应该看到：
```
右键点击！ {x: 50, y: 60}
```

### 3. 检查菜单是否出现

应该看到自定义菜单：
```
🍊 西柚桌宠
───────────
切换状态
  😊 开心
  💼 工作
  😴 休息
  🌸 待机
───────────
🚪 退出
```

---

## 🐛 如果还是不行

### 检查 1：查看控制台

按 `F12` 打开开发者工具，看是否有错误信息

### 检查 2：查看日志

右键时应该在控制台看到 "右键点击！" 的日志

### 检查 3：测试不同位置

- 点击西柚正中心
- 点击西柚上方（绿叶）
- 点击西柚周围

每次都应该触发菜单

---

## 📝 预期效果

### 左键：拖拽
- 按住左键 → 西柚变半透明
- 移动鼠标 → 窗口跟随移动
- 松开鼠标 → 恢复正常

### 右键：菜单
- 右键点击 → 自定义菜单弹出
- 点击选项 → 执行对应操作
- 点击空白 → 菜单关闭
