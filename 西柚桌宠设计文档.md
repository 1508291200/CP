## 📋 项目概述

### 项目简介
西柚桌宠是一款以西柚为主题的全功能桌面助手应用，提供三种可切换的角色形象（正常比例、Q版、纯水果），集成了工作效率管理、灵感记录、活动统计等实用功能，是你的智能工作伙伴和创意管家。

### 目标用户
需要提升工作效率、记录灵感想法、量化工作习惯的个人用户。

### 核心特性
- ✅ 三种可切换的角色形象
- ✅ 透明窗口，始终置顶
- ✅ 流畅的动画系统
- ✅ 丰富的交互反馈
- ✅ 实用的提醒功能（番茄钟、喝水提醒）
- ✅ 💡 智能灵感管理系统（快速记录、自动归类）
- ✅ 📊 活动监控统计（鼠标、键盘、应用使用追踪）
- ✅ 本地数据持久化，隐私安全

#### 3.5 📊 活动监控统计

**核心功能**
全面追踪工作活动，量化工作效率，发现使用习惯，提供数据洞察

**监控指标**
```yaml
基础指标:
  - 🖱️ 鼠标点击数（左/右/中键）
  - 🖱️ 鼠标移动距离（像素→公里）
  - ⌨️ 键盘按键数
  - ⌨️ 键盘输入字数
  - 📸 截图次数
  - 📋 复制/粘贴次数
  - ⏱️ 活跃时长（真实工作时间）
  - 😴 休息时长（空闲时间）

应用使用:
  - 应用切换次数
  - 各应用使用时长 TOP 10
  - 应用使用时长占比

文件操作（可选）:
  - 文件打开/保存次数
  - 编辑的文件类型分布
  - 代码行数统计

系统状态:
  - 启动/关机时间
  - 锁屏次数
```

```yaml
框架: Electron
```

### 为什么选择 Electron + React？

✅ **优势**
- 跨平台（Windows/Mac/Linux）
- Web 技术栈，开发迅速
- 丰富的生态系统
- 透明窗口支持完善
- 热更新，开发体验好

⚠️ **劣势与解决方案**
- 内存占用较高 → 优化打包，去除不必要的依赖
- 启动速度 → 使用 V8 快照，延迟加载非关键模块

**Animation System - 动画系统**
```typescript
// Sprite动画组件
interface SpriteAnimationProps {
  spriteSheet: string;
  animation: Animation;
  onFrame?: (frame: number) => void;
  onComplete?: () => void;
}

const SpriteAnimation: React.FC<SpriteAnimationProps> = ({
  spriteSheet,
  animation,
  onFrame,
  onComplete,
}) => {
  const [currentFrame, setCurrentFrame] = useState(0);
  
  useEffect(() => {
    const interval = setInterval(() => {
      setCurrentFrame(prev => {
        const nextFrame = (prev + 1) % animation.frames.length;
        if (nextFrame === 0 && !animation.loop) {
          clearInterval(interval);
          onComplete?.();
        }
        return nextFrame;
      });
    }, 1000 / animation.frameRate);
    
    return () => clearInterval(interval);
  }, [animation]);
  
  const frameIndex = animation.frames[currentFrame];
  const backgroundPosition = calculateSpritePosition(frameIndex);
  
  return (
    <div
      className="sprite"
      style={{
        backgroundImage: `url(${spriteSheet})`,
        backgroundPosition,
      }}
    />
  );
};
```

```typescript
// src/hooks/useDrag.ts
export const useDrag = () => {
  const [isDragging, setIsDragging] = useState(false);
  const [offset, setOffset] = useState({ x: 0, y: 0 });

  const handleMouseDown = (e: React.MouseEvent) => {
    setIsDragging(true);
    setOffset({
      x: e.clientX,
      y: e.clientY,
    });
  };

  const handleMouseMove = (e: MouseEvent) => {
    if (!isDragging) return;
    
    const deltaX = e.screenX - offset.x;
    const deltaY = e.screenY - offset.y;
    
    window.electron.window.setPosition(deltaX, deltaY);
  };

  const handleMouseUp = () => {
    setIsDragging(false);
  };

  useEffect(() => {
    if (isDragging) {
      window.addEventListener('mousemove', handleMouseMove);
      window.addEventListener('mouseup', handleMouseUp);
    }
    
    return () => {
      window.removeEventListener('mousemove', handleMouseMove);
      window.removeEventListener('mouseup', handleMouseUp);
    };
  }, [isDragging]);

  return { isDragging, handleMouseDown };
};
```

### 2. Sprite动画

```typescript
// src/components/Character/SpriteAnimation.tsx
interface Props {
  spriteSheet: string;
  frameWidth: number;
  frameHeight: number;
  frames: number[];
  frameRate: number;
  loop: boolean;
}

export const SpriteAnimation: React.FC<Props> = ({
  spriteSheet,
  frameWidth,
  frameHeight,
  frames,
  frameRate,
  loop,
}) => {
  const [currentFrameIndex, setCurrentFrameIndex] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setCurrentFrameIndex(prev => {
        const next = prev + 1;
        if (next >= frames.length) {
          return loop ? 0 : prev;
        }
        return next;
      });
    }, 1000 / frameRate);

    return () => clearInterval(interval);
  }, [frames, frameRate, loop]);

  const currentFrame = frames[currentFrameIndex];
  const col = currentFrame % (Math.floor(/* spriteSheetWidth */ 1000 / frameWidth));
  const row = Math.floor(currentFrame / (/* spriteSheetWidth */ 1000 / frameWidth));

  return (
    <div
      style={{
        width: frameWidth,
        height: frameHeight,
        backgroundImage: `url(${spriteSheet})`,
        backgroundPosition: `-${col * frameWidth}px -${row * frameHeight}px`,
        imageRendering: 'pixelated', // 像素完美渲染
      }}
    />
  );
};
```

```
4. 西柚进入待机状态，在桌面右下角陪伴
```

### 场景2：工作中突发灵感

```
1. 正在写代码，突然想到一个优化方案

4. 快速输入想法："可以用缓存优化这个API调用"
5. 系统自动识别标签 #优化 #性能
6. 自动分类为"技术"类别
7. 回车保存，西柚播放庆祝动画："灵感已保存！✨"
8. 继续工作
```

```

5. 弹出提示："工作完成！休息5分钟吧~ 💧"
6. 西柚进入"休息"状态
7. 活动监控检测到空闲，记录为休息时间
```

```
1. 活动监控检测到连续工作2小时

```
